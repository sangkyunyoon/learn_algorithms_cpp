## 시간 복잡도 O(N * logN)

[1. 100만개 정렬](https://www.acmicpc.net/problem/2751)

데이터의 갯수가 100만개인데 시간 제한이 1초으므로 반드시 시간 복잡도를 고려해야한다.

평균 시간 복잡도가 O(N * logN)인 퀵 정렬로는 불가능 최악 시간 복잡도가 O(N ^ 2)이기 때문이다.

하지만 해당 문제는 C++ STL에 sort로 해결이 가능하다.
